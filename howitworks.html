How It Works

Enter a Location

Start by entering the location you want to rank GitHub users from. It can be a city, state, or any specific location you are interested in. Note that the location of the user is determined by what is present under their location tag, so experiment with different locations naming conventions. For example "NY" rather than "New York"

Get Top Users

Click the "Go" button to retrieve the top GitHub users from the specified location. The algorithm gathers the most influential users based on their amount of followers and repositories. For now, only the top ten users are displayed due to reduce API usage.

Explore User Profiles

Once the list of top users is generated, they are ranked from most influential to least. You can explore each user's profile by clicking on their name. You'll find more details about their repositories, contributions, and more.
